# We Are Scan Me

A pet tracking and street animal welfare initiative for Koh Samui, Thailand.

## Overview

QR code-based system to help reunite lost pets with owners and track street animal welfare programs.

**Launching 2026**

## Tech Stack

- **Framework**: Next.js 14 (App Router) + TypeScript
- **Styling**: Tailwind CSS
- **Database**: Supabase PostgreSQL
- **ORM**: Prisma
- **Auth**: Supabase Auth (abstracted)
- **Storage**: Cloudinary (images)
- **Maps**: OpenStreetMap + Leaflet
- **Hosting**: Vercel

**Cost**: $0/month (free tiers)

## Key Features

- Bilingual (English/Thai)
- QR code generation with custom styling
- Location tracking on scan
- Pet owner dashboard
- Street animal welfare tracking
- Instant owner notifications

## Site Structure

- **4 English pages**: Homepage, Pet Owners, Learn More, Street Dogs & Cats
- **4 Thai pages**: Translated versions
- **Dynamic animal profiles**: `/animals/[id]`
